Nyelv Története

A JavaScript kezdetben, a 90-es évek elején, a Netscape Communications Corporation alkalmazottja, Brendan Eich dolgozott az első változaton, amelyet Mocha-ként ismertek először. Később a LiveScript nevet kapta, majd végül a JavaScript lett az elfogadott név, részben azért, hogy kihasználják a Java népszerűségét. A nyelv hirtelen népszerűvé vált a webfejlesztők körében, részben a Netscape Navigator böngésző elterjedésének köszönhetően. Azonban, a fejlődés során a böngészők közötti inkompatibilitások és a nyelv sajátosságai miatt sok kritika érte. Az igazi áttörés 2009-ben következett be, amikor a Google kiadta a Chrome böngészőt, amely a V8 JavaScript motorra épült. Ez a motor gyors és hatékony végrehajtást tett lehetővé, és azóta is az egyik legelterjedtebb JavaScript motor maradt. A JavaScript terjedését tovább fokozta a Node.js megjelenése 2009-ben, amely lehetővé tette a JavaScript szerveroldali alkalmazások futtatását is. Ezáltal a nyelv nem csak a böngészőkben, hanem a szervereken is népszerűvé vált. A nyelv folyamatos fejlődésen ment keresztül, új verziók, mint például az ECMAScript 5, 6, 7 stb., hoztak számos új funkciót és lehetőséget a fejlesztők számára. A modern JavaScript egy erőteljes és sokoldalú nyelv lett, amely a webfejlesztés szinte minden területén megtalálható.
